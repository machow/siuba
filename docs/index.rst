Siuba
=====

.. image:: siuba.svg
    :width: 400px

Siuba is a library for quick, scrappy data analysis in Python.
It is a port of
`dplyr <https://dplyr.tidyverse.org>`_,
`tidyr <https://tidyr.tidyverse.org>`_,
and other R Tidyverse libraries.
